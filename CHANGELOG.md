# Changelog

All notable changes to **uPrime** are documented in this file.

---

## [v0.2-alpha] - 2026

### Added
- Standalone Windows executable (.exe)
- Correlation analysis (spatial and temporal)
- Spatial spectral analysis
- TKE budget module
- Data export:
  - CSV for plots
  - Tecplot-compatible ASCII `.dat` for 2D fields
- Line, ROI, and full-field evaluation modes
- Module-specific parameter panels for analysis settings

### Improved
- GUI layout and workflow
- Plotting performance and usability
- Data handling for large snapshot sets

### Notes
- This is an alpha release
- Some modules are still under development
- Results depend on user-selected parameters and sample size

---

## [v0.1-alpha]

### Added
- Initial release
- Data loading for Tecplot `.dat` files
- Mean flow visualization
- Reynolds stresses ($R_{ij}$)
- Turbulent kinetic energy (TKE)
- Temporal spectral analysis (Welch PSD)
- Anisotropy invariant analysis (Lumley triangle, barycentric map)

### Notes
- Early prototype version
