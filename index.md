# uPrime: GUI-based Velocity Field Analysis Software for PIV and CFD

*Turbulence is complex. Analysis shouldn’t be.*

uPrime is an open-source, GUI-driven software for post-processing and turbulence analysis of velocity field data from Particle Image Velocimetry (PIV) and CFD.

Built from an experimental fluid mechanics perspective, uPrime integrates advanced turbulence analysis tools into a single, unified interface, eliminating the need for scripting and fragmented workflows.

## Interface

![uPrime GUI](docs/images/gui.png)

## Key Features
- Fully integrated graphical interface (no scripting required)
- Reynolds stresses and turbulent kinetic energy (TKE)
- Energy spectra and correlation analysis
- Proper Orthogonal Decomposition (POD)
- Designed for large, high-resolution, time-resolved datasets

## Get Started

- Download the latest release from GitHub
- Load Tecplot `.dat` files
- Begin analysis immediately
